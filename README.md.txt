# CoreCore Cloud Factory

Streamlit Cloud App zum Generieren von 30-Sekunden Core-Core Videos.

## Deployment auf Render

1. Neues Web Service → GitHub Repository auswählen
2. Build Command: `pip install -r requirements.txt`
3. Start Command: `streamlit run app.py --server.port $PORT --server.address 0.0.0.0`
4. Deploy → Link öffnen
